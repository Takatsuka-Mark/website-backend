## Functions Implemented:
#### Traditional FunctionFus:
| Function | Form | Description |
| --- | --- | --- |
| Addition          | a + b | Add |
| Subtraction       | a - b | Subtract |
| Multiplication    | a * b | Multiply | 
| Division          | a / b | Divide |
| Mod (remainder)   | a % b | Modulo |
| Absolute Value    | abs( a ) | Absolute Value |



## Compile Protos
```sh
mvn clean install
```
